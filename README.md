# 🧠 MERN ThinkBoard

A full-stack note-taking and idea management web application built using the **MERN stack (MongoDB, Express.js, React.js, Node.js)**.  
ThinkBoard allows users to create, organize, and manage notes efficiently with a fast and responsive interface.

---

## 🚀 Features

- 📝 **Notes Management**  
  Create, update, and delete notes  

- 📂 **Smart Organization**  
  Clean and structured notes display  

- ⚡ **Performance Optimized**  
  Efficient API handling and smooth frontend rendering  

- 🎨 **Modern UI**  
  Responsive and minimal design  

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- Axios  
- Tailwind CSS / CSS  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB Atlas  

### Tools  
- Git & GitHub  

---

##🧪 .env Setup

Backend (/backend)
MONGO_URI=<your_mongo_uri>

UPSTASH_REDIS_REST_URL=<your_redis_rest_url>
UPSTASH_REDIS_REST_TOKEN=<your_redis_rest_token>

NODE_ENV=development

##🔧 Run the Backend

cd backend
npm install
npm run dev

##💻 Run the Frontend
cd frontend
npm install
npm run dev
