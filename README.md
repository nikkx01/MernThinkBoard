# 🧠 MERN ThinkBoard

A full-stack note-taking and idea management web application built using the **MERN stack (MongoDB, Express.js, React.js, Node.js)**.  
ThinkBoard allows users to create, organize, and manage notes efficiently with a fast and responsive interface.

---

## 🚀 Features

- 📝 **Notes Management**  
  Create, update, and delete notes  

- 📂 **Smart Organization**  
  Clean and structured notes display  

- ⚡ **Performance Optimized**  
  Efficient API handling and smooth frontend rendering  

- 🎨 **Modern UI**  
  Responsive and minimal design

## 🌐 API Endpoints

| Method | Endpoint            | Description       |
|--------|--------------------|------------------|
| POST   | /api/auth/signup   | Register user    |
| POST   | /api/auth/login    | Login user       |
| GET    | /api/notes         | Get all notes    |
| POST   | /api/notes         | Create note      |
| PUT    | /api/notes/:id     | Update note      |
| DELETE | /api/notes/:id     | Delete note      |
